A Xcel CLI is a simple and lightweight command line interface for creating, editing, and viewing Excel spreadsheets.

## Installation
Step 1:
Git - `git clone https://github.com/VidyutPrabakaran1/Xcel-CLI.git`
or
Download and extract the zip file from the latest release found in the releases tab.

Step 2:
Install libraries - `pip install openpyxl xlsxwriter`
